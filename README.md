# Customer Churn Prediction (Machine Learning)

This project predicts whether a telecom customer will leave the service (churn) using machine learning.

## Dataset
Telco Customer Churn dataset.

## Tools & Libraries
- Python
- pandas
- scikit-learn
- seaborn
- matplotlib
- Jupyter Notebook

## Project Steps
1. Data loading and exploration
2. Data cleaning and preprocessing
3. Converting categorical data to numerical values
4. Splitting the dataset into training and testing sets
5. Training a Logistic Regression model
6. Evaluating the model using accuracy and confusion matrix
7. Visualizing data relationships

## Model
Algorithm used: Logistic Regression

## Results
Model accuracy: **~82%**

## Visualization
- Churn distribution
- Confusion matrix
- Tenure vs Churn analysis

## Goal
The goal of this project is to understand customer behavior and predict which customers are likely to leave the service.
